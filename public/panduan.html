Cara Melakukan Pembelian

1

Pilih Produk

Kunjungi halaman Produk atau Panel untuk melihat daftar produk yang tersedia. Pilih produk yang sesuai dengan kebutuhan Anda.

2

Hubungi Admin

Hubungi kami melalui WhatsApp atau Telegram untuk melakukan pemesanan. Sertakan nama produk yang ingin dibeli.

WhatsApp: +62882003493812 | Telegram: t.me/fathirsthore
3

Lakukan Pembayaran

Admin akan mengirimkan invoice dan nomor rekening/e-wallet untuk pembayaran. Kami menerima pembayaran via QRIS, Transfer Bank, dan E-Wallet.

4

Terima Produk

Setelah pembayaran dikonfirmasi, produk akan segera dikirimkan ke Anda. Untuk panel, data login akan dikirimkan dalam format teks.

Cara Membuat Panel Server

1

Pilih Paket Panel

Kunjungi halaman Panel dan pilih paket yang sesuai dengan kebutuhan Anda. Kami menyediakan berbagai pilihan RAM mulai dari 1GB hingga 8GB.

2

Isi Form Pembuatan

Masukkan username yang diinginkan dan pilih paket panel. Username hanya boleh mengandung huruf dan angka.

Pastikan username belum pernah digunakan di panel kami.
3

Lakukan Pembayaran

Hubungi admin untuk melakukan pembayaran. Setelah pembayaran dikonfirmasi, panel akan dibuat secara otomatis.

4

Akses Panel Anda

Data login akan dikirimkan oleh admin. Anda dapat menyalin atau mendownload data tersebut untuk disimpan dengan aman.

Cara Download Produk Digital

1

Pilih Produk

Kunjungi halaman Produk dan pilih kategori "Script/File". Lihat deskripsi dan detail produk sebelum membeli.

2

Lakukan Pembelian

Hubungi admin untuk melakukan pembelian produk yang diinginkan.

3

Download File

Setelah pembayaran dikonfirmasi, admin akan mengirimkan link download atau file langsung melalui chat.

Pastikan untuk mendownload file segera karena link memiliki batas waktu.
